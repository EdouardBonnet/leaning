\paragraph{Graph parameters.}
We use the same finite-graph-parameter interface and concrete twin-width definition as the twin-width versus mixed-minor-number submission.  Treewidth is defined by finite tree decompositions.

\paragraph{Bonnet--Depres gap.}
The Bonnet--Depres construction gives, for every natural number $k$, a finite simple graph $G$ with treewidth at most $2k+4$ and twin-width strictly greater than $2^k$.

\paragraph{Main result.}
Twin-width can be exponential in treewidth: for every $k$, there is a finite simple graph $G$ satisfying $\operatorname{tw}(G) \le 2k+4$ and $\operatorname{tww}(G) > 2^k$.
