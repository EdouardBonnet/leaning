\paragraph{Graph parameters.}
We use the graph-parameter type from the twin-width versus mixed-minor-number submission and define the twin-width parameter in this submission as that imported parameter.  We also record treewidth as a graph parameter.

\paragraph{Bonnet--Depres gap.}
The Bonnet--Depres construction gives, for every natural number $k$, a finite simple graph $G$ with treewidth at most $2k+4$ and twin-width strictly greater than $2^k$.

\paragraph{Main result.}
Consequently, twin-width can be exponential in treewidth: for every $k$, there is a finite simple graph $G$ satisfying $\operatorname{tw}(G) \le 2k+4$ and $\operatorname{tww}(G) > 2^k$.
